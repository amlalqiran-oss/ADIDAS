
# ADIDAS – Break The Limits 👟

A high-performance landing page concept for **ADIDAS**, designed to capture the brand's "Impossible Is Nothing" philosophy. This project features a bold, energetic UI focused on product innovation, legacy, and a seamless shopping experience.

[![Live Demo](https://img.shields.io/badge/Live-Demo-white)](https://amlalqiran-oss.github.io/ADIDAS/)

## 🚀 Key Features

- **Dynamic Hero Section:** A striking visual entry point featuring the 2024 collection with a call-to-action for the latest performance gear.
- **Technology Highlights:** Specialized sections showcasing proprietary innovations like **Boost Technology**, **Primeknit**, and **4D Midsole** with performance metrics.
- **Multimedia Integration:** Embedded video capabilities ("Watch Film") to enhance brand storytelling.
- **Product Categorization:** Intuitive navigation for Men, Women, Kids, and specific sports (Running, Football, Basketball).
- **Modern UI Elements:** Clean grid layouts for new arrivals, statistics tracking (70+ years of innovation), and a "Join the Movement" newsletter subscription.
- **Responsive Motion:** Designed with smooth transitions and a mobile-first approach to ensure a premium feel across all devices.

## 🛠️ Built With

- **HTML5:** Semantic markup for structured content and SEO optimization.
- **CSS3:** Advanced styling using modern layouts (Flexbox/Grid), custom animations, and bold athletic typography.
- **JavaScript:** Handles navigation menus, interactive UI components, and scroll-based interactions.
- **Unsplash API/Imagery:** High-resolution sports photography to drive visual engagement.

## 📂 Project Structure

```text
├── index.html          # Main landing page
```
